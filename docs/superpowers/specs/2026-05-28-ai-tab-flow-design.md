# AI Tab Flow 设计方案

**版本**: 1.0  
**日期**: 2026-05-28  
**状态**: 已批准

---

## 设计目标

将AI Tab重构为现代化、去AI味的全能工作台。参考Apple Intelligence + Claude + Notion AI风格，采用极简留白设计，对话以文档流形式呈现。

---

## 核心设计原则

1. **内容即界面** — 无气泡、无头像、无渐变，对话如文档流
2. **极简留白** — 大量空白，信息密度适中，视觉负担最小
3. **零硬编码颜色** — 全部跟随Material3主题，支持Material You动态取色
4. **功能即工作流** — 对话、Agent、知识库三种模式无缝切换

---

## 设计禁忌（明确不做）

- ❌ 渐变彩虹/紫蓝色调（典型"AI紫"、"科技蓝"）
- ❌ 卡通/拟物风格（可爱机器人图标、卡通UI）
- ❌ 圆角气泡+头像（微信聊天式对话）
- ❌ 过多图标和按钮（工具栏堆砌）

---

## 整体布局架构

```
┌─────────────────────────────┐
│  [对话]  [Agent]  [知识]  ⚙  │  ← 顶部：pill切换器 + 设置
├─────────────────────────────┤
│                             │
│                             │
│      (内容滚动区域)          │  ← 占满剩余空间，maxWidth 640dp居中
│                             │
│                             │
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │ 输入问题...        ●  │  │  ← 底部：浮动输入栏
│  └───────────────────────┘  │
└─────────────────────────────┘
```

### 顶部栏
- 左侧：3个pill按钮（对话/Agent/知识）
  - 选中态：`onSurface`填充 + `surface`色文字，圆角16dp，高度32dp，padding水平14dp
  - 未选中态：透明背景 + `onSurfaceVariant`文字
  - 切换动画：选中背景滑动过渡（`animateColorAsState`，200ms）
- 右侧：设置齿轮图标（`IconButton`，`onSurfaceVariant`色）
- 无标题、无阴影、无分割线
- 高度48dp，padding水平16dp

### 内容区
- `LazyColumn`，占满中间全部空间
- 欢迎态和对话态通过`Crossfade`切换
- 内容最大宽度640dp（平板适配），居中显示

### 输入栏
- 贴底浮动，圆角pill形状（28dp圆角）
- 背景：`surfaceVariant`
- 发送按钮：24dp圆形，`onSurface`填充
- 高度48dp，水平margin 16dp，底部padding 12dp

## 欢迎页（空状态）

```
┌─────────────────────────────┐
│  [对话]  [Agent]  [知识]  ⚙  │
├─────────────────────────────┤
│                             │
│          你好               │  ← displaySmall，Bold
│     有什么我可以帮你的       │  ← bodyLarge，onSurfaceVariant
│                             │
│  ─────────────────────────  │
│                             │
│  → 总结今天的笔记            │  ← bodyMedium，文字建议
│  ─────────────────────────  │
│  → 帮我整理待办事项          │
│  ─────────────────────────  │
│  → 搜索相关知识              │
│  ─────────────────────────  │
│                             │
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │ 输入问题...        ●  │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

### 问候语
- 垂直居中偏上（约40%位置）
- "你好"：`displaySmall`，`onSurface`，fontWeight Bold
- 副标题：`bodyLarge`，`onSurfaceVariant`，marginTop 4dp
- 根据时间段变化："早上好"/"下午好"/"晚上好"

### 建议列表
- 问候语下方，间距24dp
- 每条建议：纯文字行，左侧"→"箭头符号（`onSurfaceVariant`色）
- 行间用`HorizontalDivider`（`outline`色，alpha 0.3）分隔
- 点击后直接填入输入框并发送
- 最多显示4条，建议内容可配置

## 对话区（消息展示）

```
┌─────────────────────────────┐
│  [对话]  [Agent]  [知识]  ⚙  │
├─────────────────────────────┤
│                             │
│  帮我总结一下今天的工作笔记   │  ← 用户消息：纯文本，右对齐
│                             │
│  │ 今日工作摘要               │  ← AI回复：左侧2px竖线
│  │                           │    surfaceVariant背景，圆角8dp
│  │ 完成了3个API接口开发，     │
│  │ 修复了2个UI bug...        │
│  │                           │
│  │ [复制]  [插入笔记]         │  ← 操作按钮：labelSmall
│                             │
│  把摘要保存到今天的日记里     │  ← 用户消息
│                             │
│  ┌ ● 已保存到「01-15 日记」┐ │  ← 操作结果：内联状态卡片
│  └─────────────────────────┘ │
│                             │
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │ 继续对话...        ●  │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

### 用户消息
- 纯文本，`bodyMedium`，`onSurface`
- 无气泡、无背景色、无头像
- 右对齐，maxWidth 80%
- 间距：上下各12dp

### AI回复
- 左侧2px竖线（`outline`色），作为唯一视觉标识
- 背景：`surfaceVariant`，圆角8dp，padding 14dp
- 文字：`bodyMedium`，`onSurface`，行高1.7
- 左对齐，maxWidth 90%
- 底部操作栏：`[复制]` `[插入笔记]` 两个TextButton，`labelSmall`，`onSurfaceVariant`色
- 间距：上下各12dp

### 操作结果卡片（Agent执行、保存文件等）
- 单行内联卡片，高度约36dp
- 背景：`surfaceVariant`，圆角10dp
- 左侧小圆点（绿色=成功，红色=失败）+ 文字描述
- 无图标，极简

### 思考中状态
- 3个小圆点脉冲动画（`outline`色），替代当前的大面积加载动画
- 左侧同样有2px竖线，保持视觉一致

### 联网搜索结果
- 折叠式卡片："联网搜索 · 5条结果"，点击展开
- 展开后每条结果：标题+来源域名，纯文字列表

### 打字机效果
- 保留现有帧同步实现（120字符/秒）
- 仅在AI回复最后一条消息上生效

## 输入栏

```
单行状态：
┌───────────────────────────┐
│  输入问题...           ●  │
└───────────────────────────┘

多行展开：
┌───────────────────────────┐
│  帮我写一份项目计划书，    │
│  包含时间安排和人员分工    │
│                        ●  │
└───────────────────────────┘

生成中状态：
┌───────────────────────────┐
│  ■  正在生成...            │  ← 发送按钮变为停止按钮
└───────────────────────────┘
```

### 容器
- `Surface`，圆角28dp，背景`surfaceVariant`
- 水平margin 16dp，底部padding 12dp
- 最小高度48dp，最大高度120dp（超出可滚动）
- 无阴影、无边框

### 输入框
- `BasicTextField`，`bodyMedium`，`onSurface`
- placeholder："输入问题..."，`onSurfaceVariant`色，alpha 0.6
- 自动扩展高度，最多4行后内部滚动

### 发送按钮
- 右侧对齐，24dp圆形
- 有内容时：`onSurface`填充 + 白色箭头图标
- 无内容时：`outline`色填充，禁用态
- 生成中时：变为停止按钮（方形图标），`error`色

### 附加功能按钮
- 联网搜索：小文字标签"联网"，选中时`primary`色+下划线
- 无其他图标按钮，保持极简

### 键盘适配
- `WindowInsets.ime` 自动跟随键盘弹出
- 输入栏始终在键盘上方，无遮挡

## 模式切换与设置

### 顶部pill切换器
- 3个pill按钮水平排列，间距4dp
- 选中态：`onSurface`填充 + `surface`色文字，圆角16dp，高度32dp，padding水平14dp
- 未选中态：透明背景 + `onSurfaceVariant`文字
- 切换动画：选中背景滑动过渡（`animateColorAsState`，200ms）
- 右侧齿轮图标：`IconButton`，`onSurfaceVariant`色

### 设置面板（ModalBottomSheet）
```
┌─────────────────────────────────┐
│  ───  (拖拽条)                   │
│                                 │
│  模型                            │
│  ┌─────────────────────────┐    │
│  │ GPT-4o              ▾  │    │  ← 下拉选择
│  └─────────────────────────┘    │
│                                 │
│  搜索引擎                        │
│  ┌─────────────────────────┐    │
│  │ Bing                ▾   │    │
│  └─────────────────────────┘    │
│                                 │
│  对话历史                        │
│  [清除所有对话]                  │  ← 红色文字按钮
│                                 │
└─────────────────────────────────┘
```

- 替代当前的`CollapsibleTopBar`中的模型选择+更多菜单
- 所有设置集中在一个Sheet中，不再有多余的顶栏按钮
- Sheet圆角顶部28dp，背景`surface`

### Agent模式差异
- 输入栏placeholder变为"描述你要执行的任务..."
- AI回复中Agent步骤以内联文本显示（"思考中..."→"执行: 搜索网页"→"完成"），不使用当前的`AgentStepCard`大卡片

### 知识库模式差异
- 输入栏placeholder变为"搜索笔记..."
- 结果以文档列表形式展示，每条结果：标题+摘要+时间，纯文字行

## 配色规范

完全跟随Material3主题，零硬编码颜色：

| 元素 | 亮色 | 暗色 | 来源 |
|------|------|------|------|
| 页面背景 | `#F8F9FA` | `#1C1B1F` | `background` |
| AI回复背景 | `#F1F3F4` | `#3C3A41` | `surfaceVariant` |
| 输入栏背景 | `#F1F3F4` | `#3C3A41` | `surfaceVariant` |
| 用户消息文字 | `#1C1B1F` | `#E6E1E5` | `onSurface` |
| AI竖线/分割线 | `#DADCE0` | `#49454F` | `outline` |
| 次级文字 | `#49454F` | `#CAC4D0` | `onSurfaceVariant` |
| Pill选中态 | `#1C1B1F` | `#E6E1E5` | `onSurface` |
| 成功状态点 | `#34C759` | `#30D158` | 系统绿 |
| 错误状态点 | `#FF3B30` | `#FF453A` | 系统红 |

**关键原则**：删除ChatBubble中所有硬编码颜色（`UserBubbleColor #10A37F`、`AIBgColor #FFFFFF`等），全部改用`MaterialTheme.colorScheme`。支持Material You动态取色。
