NSAI Notes Privacy Policy

Last updated: April 2026

1. Information Collection

NSAI Notes does not collect any personally identifiable information. All note data, tags, and app settings are stored locally on your device.

2. AI Features

AI features require you to configure your own third-party AI service API Key. When using AI features, note content is sent to your chosen AI service provider for processing. We do not log or store these requests.

3. Data Storage

All data is encrypted using SQLCipher and stored in the device's local database. API Keys are protected using Android Keystore hardware-level encryption.

4. Permissions

This app only requests the following necessary permissions:

  • Microphone: Used for voice-to-text, requested only when actively used
  • Internet: Used for AI API communication

5. Third-Party Services

AI features connect to the following third-party services via your configured API Key:

  • DeepSeek API
  • Kimi (Moonshot) API
  • GLM (Zhipu) API
  • MiniMax API

Data processing by these services is governed by their respective privacy policies.

6. Data Security

We use industry-standard security measures to protect your data, including local database encryption and secure API key storage.

7. Children's Privacy

This app is not directed at children under 13 and does not knowingly collect personal information from children.

8. Contact

For privacy-related questions, please contact us through the in-app feedback feature.
