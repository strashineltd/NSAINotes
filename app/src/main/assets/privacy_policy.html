NSAI笔记 隐私政策

最后更新日期：2026年4月

1. 信息收集

NSAI笔记 不收集任何个人身份信息。所有笔记数据、标签和应用设置仅存储在您的设备本地。

2. AI功能

AI功能需要您自行配置第三方AI服务的API Key。当您使用AI功能时，笔记内容会被发送到您所选的AI服务提供商进行处理。我们不会记录或存储这些请求。

3. 数据存储

所有数据均使用SQLCipher加密存储在设备本地数据库。API Key使用Android Keystore系统进行硬件级加密保护。

4. 权限说明

本应用仅申请以下必要权限：

  • 麦克风权限：用于语音转文字功能，仅在您主动使用时请求
  • 网络权限：用于AI API通信

5. 第三方服务

AI功能通过您配置的API Key连接以下第三方服务：

  • DeepSeek API
  • Kimi (Moonshot) API
  • GLM (智谱) API
  • MiniMax API

这些服务的数据处理受其各自的隐私政策约束。

6. 数据安全

我们采用行业标准的安全措施保护您的数据，包括本地数据库加密和安全的API Key存储。

7. 儿童隐私

本应用不面向13岁以下儿童，不会故意收集儿童的个人信息。

8. 联系我们

如有隐私相关问题，请通过应用内反馈功能联系我们。
